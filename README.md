# MeghDrishti

MeghDrishti is a real-time weather web application that provides accurate and location-based forecasts using external APIs. It includes an interactive map, fast location search, pincode support, and weather-based music playback for a more engaging experience.

## Features

• Real-time weather updates from OpenWeather  
• Location search for cities, towns, villages and localities  
• Interactive map using Leaflet  
• Automatic weather detection via geolocation  
• Music that adapts based on weather conditions  
• Responsive and user-friendly interface

## Technologies Used

• HTML, CSS, JavaScript  
• Leaflet  
• OpenWeather API  
• Nominatim Geocoding API

## Setup

1. Clone the repository  
   ```bash
   git clone https://github.com/tusharrr0302/MeghDrishti
2. Add your OpenWeather API key in script.js  
3. Run the project using a local server or open home.html directly
